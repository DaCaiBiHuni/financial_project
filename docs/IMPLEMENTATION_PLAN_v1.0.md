# 投资组合追踪桌面应用
## 一期开发计划 v1.0

---

# 1. 开发目标

在第一期完成一个可运行的桌面原型，支持：
- 产品管理
- 投资组合查看
- 本地存储
- 基础图表展示

---

# 2. 阶段划分

## 阶段 1：项目初始化
- 建立项目结构
- 初始化桌面应用主窗口
- 建立 docs / src / data 目录
- 建立 SQLite 数据库连接

## 阶段 2：产品模块
- 设计 Product 数据模型
- 完成产品列表页
- 完成添加产品表单
- 完成产品详情页

## 阶段 3：组合模块
- 设计 Position 数据模型
- 完成持仓录入
- 完成持仓列表
- 完成组合汇总计算

## 阶段 4：图表与仪表盘
- 完成 Dashboard 页面
- 接入产品价格趋势图
- 显示总收益、总市值等核心指标

## 阶段 5：设置与整理
- 设置页基础结构
- 数据源配置预留
- 项目结构整理
- 补测试与说明文档

---

# 3. 页面清单

- Dashboard 页面
- Products 页面
- Product Detail 页面
- Portfolio 页面
- Settings 页面

---

# 4. 优先级

## P0
- 主窗口
- 数据库初始化
- 产品列表
- 添加产品
- 持仓录入
- 组合总览

## P1
- 产品详情
- 趋势图
- 设置页基础结构

## P2
- 搜索与筛选
- 更多图表
- 多数据源支持

---

# 5. 第一阶段交付标准

满足以下条件即可视为一期可交付：
- 应用可启动
- 可添加产品
- 可录入持仓
- 可看到组合总览
- 数据可保存在本地
- 代码结构具备后续扩展能力

---

# 6. 下一阶段衔接

一期完成后，进入第二阶段：
- 外部数据源接入
- 历史价格同步
- 更完整的组合分析
- 交易记录模型
