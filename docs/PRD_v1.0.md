# 投资组合追踪桌面应用
## 产品需求文档（PRD）v1.0

---

# 1. 项目背景

个人投资者通常会同时关注多个投资产品，并希望持续了解这些产品的价格走势、当前持仓情况以及整体投资组合的表现。现有工具往往存在以下问题：

- 信息分散，多个平台和产品数据无法统一管理
- 更偏“看行情”，缺少“看组合”的统一视图
- 对个人用户的桌面使用场景支持不足
- 可扩展性不强，后续添加新产品、新数据源、新分析能力成本高

因此，需要开发一款桌面级投资组合追踪应用，帮助用户在本地统一管理关注产品、持仓信息和组合表现，并支持后续功能扩展。

---

# 2. 产品目标

1. 支持追踪特定投资产品走势
2. 支持展示当前投资组合详情
3. 支持快速添加新的投资产品
4. 提供良好的桌面级使用体验
5. 为后续数据源、分析能力和资产类别扩展预留空间

---

# 3. 目标用户

- 有个人投资需求的普通投资者
- 需要长期跟踪多个产品的人
- 希望统一查看关注产品与当前持仓的人
- 倾向于使用本地桌面软件管理资产的人

---

# 4. MVP 范围

## 必做功能
- 产品列表
- 添加产品
- 产品详情
- 产品价格走势
- 持仓录入
- 投资组合总览
- 本地数据存储
- 设置页基础结构

## 暂不纳入 MVP
- 自动交易执行
- 云同步
- 多账户系统
- 高级量化分析
- 社交能力

---

# 5. 核心模块

## 5.1 Dashboard
- 总成本
- 总市值
- 总收益
- 收益率
- 最近更新摘要

## 5.2 Products
- 产品列表
- 添加/编辑/删除产品
- 产品详情
- 价格走势图

## 5.3 Portfolio
- 当前持仓
- 成本价
- 当前价
- 盈亏
- 组合汇总

## 5.4 Settings
- 数据源配置
- 刷新频率
- 显示偏好

---

# 6. 非功能要求

- 可扩展
- 可维护
- 本地优先
- 界面清晰
- 常规使用场景下响应流畅

---

# 7. 版本目标

v1.0 目标：
做出一个可本地运行、可录入持仓、可追踪产品、可展示组合概况的桌面原型。
